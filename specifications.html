Specifications

I●TA is built around Nordic's nRF9160 SiP and nRF52840 SoC

Microcontroller

Using nRF9160 allows I●TA to deliver:

Arm® Cortex®-M33

  • Data watchpoint and trace (DWT), embedded trace macrocell (ETM), and instrumentation trace macrocell (ITM)
  • Serial wire debug (SWD)
  • Trace port
  • 1 MB flash
  • 256 kB low leakage RAM
  • Arm TrustZone®
  • Arm CryptoCell™ 310
  • Up to 4x SPI master/slave with EasyDMA
  • Up to 4x I2C compatible two-wire master/slave with EasyDMA
  • Up to 4x UART (CTS/RTS) with EasyDMA
  • I2S with EasyDMA

Other

  • Digital microphone interface (PDM) with EasyDMA
  • 4x pulse width modulator (PWM) unit with EasyDMA
  • 12-bit, 200 ksps ADC with EasyDMA - eight configurable channels with programmable gain
  • 3x 32-bit timer with counter mode
  • 2x real-time counter (RTC)
  • Programmable peripheral interconnect (PPI)

LTE Modem

The LTE modem inside nRF9160 packages:

Features

  • Transceiver and baseband
  • 3GPP LTE release 13 Cat-M1 and Cat-NB1 compliant
    • 3GPP release 13 coverage enhancement
  • 3GPP LTE release 14 Cat-NB2 compliant
  • GPS receiver
    • GPS L1 C/A supported
    • QZSS L1 C/A supported
  • Power saving features: DRX, eDRX, PSM
  • IP v4/v6 stack
  • Secure socket (TLS/DTLS) API

Bands

  • Cat-M1: B1, B2, B3, B4, B5, B8, B12, B13, B18, B19, B20, B25, B26, B28, B66
  • Cat-NB1/NB2: B1, B2, B3, B4, B5, B8, B12, B13, B17, B19, B20, B25, B26, B28, B66

Bluetooth

By including nRF52840 I●TA adds the following capabilities:

Radio

  • 2.4 GHz Transceiver
  • 2 Mbps, 1 Mbps, Long Range
  • Bluetooth Low Energy, Bluetooth mesh
  • ANT, 802.15.4, Thread, Zigbee
  • NFC-A
  • +8 dBm TX Power

MCU

  • 64 MHz Cortex-M4 with FPU
  • 1 MB Flash, 256 KB RAM
  • PWM
  • 12-bit ADC
  • USB 2.0

Periphery

I●TA also packs the following periphery:

Periphery

  • Low Power 3-axis accelerometer lsm303agr
    • 3 magnetic field channels and 3 acceleration channels
    • ±50 gauss magnetic dynamic range
    • ±2/±4/±8/±16 g selectable acceleration full scales
    • Single measurement mode for magnetometer
    • Programmable interrupt generators for free-fall, motion detection, and magnetic field detection
    • Embedded self-test
    • Embedded temperature sensor
  • LEDs
  • General Purpose Button

Power

  • Battery voltage reading
  • Can be powered from any combination of USB, Battery, or VIN (e.g. solar panel)
  • Nominal Operating Input voltage: 4.35V - 5.5V
  • Battery voltage: 3.2V - 4.2V
  • Over-voltage protection
  • Under-voltage protection
  • Low-noise, low power, switching power supply, with output up to 1000mA
  • USB-friendly Li-Ion/LiPolymer charger with MPPT
